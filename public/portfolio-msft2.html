Overview

I am back this summer for a second internship at Microsoft. After hearing that a commercial version of the prototype we made last summer was actually shipped to end users, I decided to accept my return offer and come back to the same team, OS Security, to keep working in the Natural Authentication space.

The ultimate goal of Natural Auth is to eliminate the need for long and complicated (and hard to remember) passwords. Instead, the computer should be able to decide (securely) whether or not you are there based on passive signals it can gather. For example, Windows Hello uses facial recognition to sign you in. My project this summer is to extend the Natural Auth framework to allow the detection of additional signals. Once done, my code will be able to get incorporated into the production code base in a few months, which is really exciting! :)

Since I'm wrapping the project up, I'm looking into another project that builds more directly on what we did last summer, Dynamic Lock. In this new project, I'll be able to work more closely with my PM and the UX team to explore the possibilty of improving the UX of the Dynamic Lock experience. Stay tuned!
